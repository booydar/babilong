# Aegyx 0.1 BABILong Submission

Aegyx 0.1 is a closed research prototype of the Aegyx system.

This directory contains BABILong QA1-QA5 prediction files for the public splits from `0k` through `10M`.
The submitted prediction files use the standard BABILong CSV format with `target`, `output`, and `question` columns.

Implementation details are not disclosed in this public result submission.
